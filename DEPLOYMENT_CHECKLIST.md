# 📋 ARKIVE Deployment Checklist

Use this checklist to ensure a smooth deployment to production.

- [ ] Create a GitHub account (if you don't have one)
- [ ] Create a new empty repository named `arkive` on GitHub
- [ ] Push local code to GitHub origin
- [ ] Create a Vercel account using your GitHub login
- [ ] Import the `arkive` repository into Vercel
- [ ] Ensure Framework Preset is set to `Vite`
- [ ] Click "Deploy"
- [ ] Verify the live URL
- [ ] Send `TEAM_QUICK_START.md` to your team members
