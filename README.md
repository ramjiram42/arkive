# 🗄️ ARKIVE

Enterprise-grade document management and asset archival system.

## Features

- **Smart Organization:** Home, Starred, Recent, Shared, and Archive views.
- **File Management:** List and Grid views with sortable metrics.
- **Dark Mode:** Easy on the eyes with automatic system theme matching.
- **Fast Search:** Instant client-side filtering and sorting.
- **Modern UI:** Built with React, Vite, and Lucide icons.

## Local Development

1. Run `npm install`
2. Run `npm run dev`
3. Open `http://localhost:5173`
