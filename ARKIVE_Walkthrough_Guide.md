# ARKIVE Walkthrough Guide

## Introduction
Welcome to the ARKIVE platform. This guide explains how the application is architected and how to use the different sections of the UI.

## Components Breakdown
- **ArkiveLogo**: The branding for the platform.
- **DocumentCard & FolderCard**: Represent files and directories in the grid view.
- **FilterPanel & SavedSearchPanel**: Help you narrow down the exact document you need.
- **StatCard**: Displays quick metrics on how many files you have.
