## 🖱️ How to Use

### Basic Navigation

1. **Browse Documents**
   - Click on **Home** in the sidebar
   - View all document classes in table/grid view

2. **Search for Documents**
   - Use the search bar at the top
   - Click on suggestions to drill down

3. **View Document Details**
   - Click on any document class name
   - See all files within that class

4. **Filter Documents**
   - Click the **Filter** button in header
   - Set your criteria
   - Click Apply

5. **Sort Columns**
   - Click any column header to sort
   - Click again to reverse order
   - Look for ↑↓ arrows

### Advanced Features

**Starred Documents** ⭐
- View your starred items: Click **Starred** in sidebar

**Recent Activity** 🕒
- See recently accessed files: Click **Recent** in sidebar
- Or check the "Recent Activity" section on home

**Shared Documents** 👥
- View shared files: Click **Shared** in sidebar

**Download Files**
- Click the three dots (•••) menu on any row
- Select **Download**

**View Statistics**
- Auto-collapsing stat cards show:
  - Total Documents (12,847)
  - Storage Used (847 GB)
  - Shared Files (3,421)
  - Active Users (156)
